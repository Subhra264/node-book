# NodeBook
This is a social media application. User can sign up, sign in, create post, see posts etc. Also it provides rich text editing experience to the users using the CKEditor. 
## Technology used -
<ol>
    <li> MERN stack </li>
    <li> ck-editor </li>
    <li> Material UI </li>
    <li> JWT token </li>
</ol>

This is still under development.
